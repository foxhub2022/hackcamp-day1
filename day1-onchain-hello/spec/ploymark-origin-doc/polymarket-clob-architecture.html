Polymarket CLOB Client V2 Architecture

@polymarket/clob-client-v2 · TypeScript SDK · submodule: polymarket-clob/

核心入口 ClobClient

  • • 单一 Facade，聚合 REST 调用、鉴权、下单全流程
  • • 构造时注入 host / chain / signer / creds
  • • 公开端点：getMarkets、getOrderBook、getPrice 等
  • • 交易端点：createAndPostOrder、cancelOrder、getTrades
  • • 内置 tickSize / negRisk / feeRates 缓存

双层鉴权模型

  • • L1：钱包 EIP-712 签名（ClobAuthDomain）创建/派生 API Key
  • • L2：HMAC-SHA256 签名每个受保护 REST 请求
  • • headers/ 组装 POLY_* 请求头
  • • 支持 EOA / POLY_PROXY / POLY_1271 签名类型
  • • Builder API Key 用于 builder 费率与 trades

订单构建与链上签名

  • • OrderBuilder → helpers/ 计算价格、数量、舍入
  • • 支持 Exchange V1/V2/V3 及 negRisk 合约路由
  • • ExchangeOrderBuilder 生成 EIP-712 Order 结构并签名
  • • 限价单 GTC/GTD + 市价单 FOK/FAK
  • • config.ts 维护 Polygon(137) / Amoy(80002) 合约地址
